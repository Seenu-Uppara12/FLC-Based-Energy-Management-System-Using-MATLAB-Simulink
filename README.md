
# FLC-Based Energy Management System Using MATLAB/Simulink

## 📌 Project Overview
This project focuses on the design and simulation of a Fuzzy Logic Controller (FLC)-based Energy Management System for efficient power distribution in renewable energy systems.

## ⚙️ Tools & Technologies
- MATLAB (R2024a)
- Simulink
- Fuzzy Logic Toolbox

## 🔍 Objectives
- To develop an intelligent energy management system
- To optimize power flow between sources
- To improve system efficiency using fuzzy logic control

## 📁 Project Structure
- `Simulation/` – Contains all simulation models and files  
- `BASE FILES/` – Base/reference models  
- `DG_SOLAR/` – Distributed generation solar system files  

## ▶️ How to Run
1. Open MATLAB
2. Navigate to the project folder
3. Open `.slx` or `.m` files
4. Run the simulation

## 📊 Results
- Improved energy efficiency
- Stable power distribution
- Better performance compared to conventional methods
